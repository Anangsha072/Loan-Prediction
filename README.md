##Loan Prediction Approval App##

##Description:
A Streamlit web application that predicts whether a loan will be approved or rejected based on user-provided financial and personal details. Users can input parameters such as income, loan amount, credit score, number of dependents, education, employment status, loan duration, and assets to get an instant prediction.

##Features:

Predicts loan approval status in real-time.

User-friendly interface built with Streamlit, featuring sliders, input fields, and dropdowns.

Interactive and visually appealing design (dark mode).

Uses a pre-trained machine learning model for predictions.

Provides immediate feedback to users (e.g., “Loan Approved” or “Loan Rejected”).

##Tech Stack:

Python

Streamlit

Pandas, NumPy

Scikit-learn (ML model)

Pickle (for model serialization)

##How to Use:

Clone the repository.

Install dependencies:

pip install -r requirements.txt


##Run the app:

streamlit run app.py


Enter your financial and personal details in the app.

Click Predict to see if the loan is likely to be approved.
